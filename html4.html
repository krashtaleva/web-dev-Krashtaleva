Paragraphs

A paragraph (from the Greek paragraphos, “to write beside” or “written beside”) is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.

[Top]

Blockquotes

A block quotation (also known as a long quotation or extract) is a quotation in a written document, that is set off from the main text as a paragraph, or block of text.

It is typically distinguished visually using indentation and a different typeface or smaller size quotation. It may or may not include a citation, usually placed at the bottom.

What is selector specificity and how is it calculated?

Selector specificity is an indicator that determines which style will be applied to an element when there are several styles that conflict with each other.

Specificity is calculated based on a combination of selector types, such as element selectors, class selectors, and identifiers.

The more precise and specific the selector, the higher its specificity and priority when applying styles.

What are complex selectors?

Complex selectors are combinations of multiple elements in a CSS rule to precisely select a target element or elements.

They allow you to precisely select elements based on various criteria such as nesting, classes, pseudo-classes and other attributes.

What is the difference between pseudo classes and pseudo elements?

Pseudo-classes apply styles to an element based on its state (for example, :hover, :active, :focus).

Pseudo-elements are used to style parts of an element (for example, ::before, ::after) or individual pieces of content (for example, ::first-line).
